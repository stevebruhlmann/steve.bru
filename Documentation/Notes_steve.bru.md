# Commandes Git

```bash
# Synchroniser avec GitHub (récupérer les modifications)
git pull

# Se placer dans le dossier du site
cd ~/Documents/20_steve.bru/site

# Envoyer les modifications sur GitHub
cd ~/Documents/20_steve.bru/site
git add -A
git commit -m "update liens sections index.html"
git push
```
